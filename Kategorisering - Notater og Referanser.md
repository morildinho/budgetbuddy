# KIWI Kategorisering - Notater og Referanser

## Kategorier og deres klassifiseringer

### Etablerte kategorier fra første kvitteringer:

| Kategori | Beskrivelse | Eksempler |
|----------|-------------|----------|
| **Kjøtt** | Kjøtt og kjøttprodukter | Kyllingfilet, kjøttboller, kjøtdeig, karbonader |
| **Fisk** | Fisk og fiskeprodukter | Laksefilet, fiskegrateng |
| **Grønnsaker** | Grønnsaker og poteter | Tomat, paprika, agurk, spinat, poteter |
| **Frukt** | Frisk frukt | Epler, bananer, appelsiner |
| **Brød** | Brød og bakervarer | Brød, rundstykker, tortilla, naan |
| **Melkeprodukter** | Melk, smør, ost | Melk, smør, revet ost |
| **Egg** | Egg i alle varianter | Egg 6-pakk |
| **Godteri** | Sjokolade og godteri | Smågodt, karamell |
| **Snacks** | Snacks og chips | Svetgull salt & pepper |
| **Drikke** | Brus, juice, vann | Coca-Cola Zero |
| **Krydder** | Krydderblandinger | Indisk Tikka Masala |
| **Kaffe** | Kaffe og kaffeprodukter | Friele Frokost filterkaffe |
| **Pålegg** | Pålegg og smørepølse | Leverpostei |
| **Mat** | Ferdigmat, konserver, pasta | Tagliatelle, pastasaus |
| **Annet** | Diverse | Pant |

## Viktige notater for fremtiden

### Kaffe
- **Friele Frokost Filterma** → **Kaffe** (ikke Godteri)
- Alle kaffeprodukter skal klassifiseres som Kaffe

### Fisk
- **Fiskegrateng** → **Fisk** (ikke Mat)
- Alle fiskeprodukter skal klassifiseres som Fisk, ikke Mat

### Pålegg
- **Leverpostei** → **Pålegg** (ikke Mat)
- Alle påleggsprodukter skal klassifiseres som Pålegg, ikke Mat

### Mat
- **Mat**-kategorien skal brukes for:
  - Pasta og pastasaus
  - Tagliatelle
  - Annen ferdigmat som ikke er fisk
